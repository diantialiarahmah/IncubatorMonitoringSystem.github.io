Tentang Smart Incubator Monitoring System

Smart Incubator Monitoring System adalah sistem pemantauan cerdas untuk mengawasi kualitas suhu dan kelembapan inkubator bayi secara real-time menggunakan Arduino Uno, sensor DHT11, dan buzzer.

Sensor DHT11 membaca kondisi lingkungan di dalam inkubator dan data diproses oleh Arduino Uno. Apabila suhu atau kelembapan berada di luar rentang aman, sistem akan mengaktifkan buzzer sebagai alarm peringatan dini.

Sistem ini dirancang sebagai solusi sederhana, ekonomis, dan andal yang dapat dikembangkan lebih lanjut ke sistem monitoring terpusat berbasis ROS 2 dalam ekosistem Smart Hospital.

Pentingnya Smart Incubator Monitoring System

Pentingnya Pengaturan Suhu dan Kelembapan

Bayi, terutama bayi prematur, belum mampu mengatur suhu tubuhnya sendiri dengan baik. Oleh karena itu, inkubator berfungsi untuk menjaga suhu dan kelembapan yang stabil agar bayi tetap hangat, nyaman, dan aman. Kondisi lingkungan yang tepat membantu bayi bernapas dengan baik, menjaga kesehatan kulit, serta mendukung proses tumbuh kembangnya.

Risiko Kondisi Tidak Stabil

Suhu dan kelembapan di dalam inkubator tidak terjaga dengan baik, bayi dapat mengalami berbagai masalah kesehatan. Suhu yang terlalu dingin dapat membuat bayi kedinginan, sedangkan suhu yang terlalu panas dapat menyebabkan bayi tidak nyaman dan kehilangan cairan tubuh. Kelembapan yang tidak sesuai juga dapat memicu gangguan pernapasan, iritasi kulit, hingga meningkatkan risiko infeksi.

Keterbatasan Pemantauan Manual

Pemantauan inkubator secara manual biasanya dilakukan dengan pengecekan berkala oleh tenaga medis. Cara ini memiliki keterbatasan karena kondisi inkubator bisa berubah kapan saja. Jika perubahan tersebut tidak segera diketahui, respons penanganan bisa terlambat. Selain itu, pemantauan manual sangat bergantung pada ketelitian manusia, sehingga berisiko terjadi kesalahan.

Alasan Sistem Otomatis

Sistem otomatis dipilih agar kondisi inkubator dapat dipantau secara terus-menerus tanpa henti. Dengan bantuan mikrokontroler dan sensor, suhu serta kelembaban dapat dipantau secara otomatis. Ketika kondisi tidak normal, sistem akan langsung memberikan peringatan melalui alarm, sehingga tindakan dapat segera dilakukan. Sistem ini sederhana, terjangkau, dan membantu meningkatkan keamanan bayi.

Fitur Penting Sistem

Real-Time Temperature & Humidity Monitoring

Sistem memantau suhu dan kelembapan inkubator secara terus-menerus menggunakan sensor DHT11, sehingga kondisi lingkungan bayi selalu terkontrol.

Automatic Alarm System (Buzzer Alert)

Buzzer akan aktif secara otomatis ketika suhu atau kelembapan berada di luar batas aman, memberikan peringatan cepat tanpa perlu intervensi manual.

Centralized Monitoring (Multi-Inkubator Ready)

Sistem dapat dikembangkan untuk memantau lebih dari satu inkubator dalam satu jaringan ROS 2, cocok untuk lingkungan rumah sakit.

ROS 2 Publisher–Subscriber Integration

Data suhu dan kelembapan dipublikasikan sebagai topic ROS 2, yang akan mempermudah Monitoring terpusat, memperbanyak subscriber (dashboard, server, nurse station) dan Komunikasi yang stabil dan real-time

Arsitektur Sistem

  • Sensor DHT11: Sensor DHT11 berfungsi sebagai perangkat input yang membaca kondisi suhu dan kelembapan di dalam inkubator bayi secara berkala.
  • Arduino Uno: Arduino Uno bertindak sebagai pusat pemrosesan data, menerima data dari sensor DHT11 dan mengolahnya sesuai dengan program yang telah ditentukan.
  • Threshold Check: Data suhu dan kelembapan yang diterima akan dibandingkan dengan batas aman yang telah ditetapkan untuk menentukan apakah kondisi inkubator berada dalam keadaan normal atau tidak.
  • Buzzer: Menunjukkan kondisi tidak normal, Arduino Uno akan mengaktifkan buzzer sebagai alarm peringatan agar pengguna segera melakukan tindakan.
  • Sistem Monitoring Berulang: Sistem bekerja secara otomatis dan terus-menerus, melakukan pembacaan, pemrosesan, dan evaluasi data tanpa perlu pengoperasian manual.

Smart Incubator Monitoring System

1️⃣Menyiapkan Sistem Operasi Utama (Windows 11)

Windows 11 digunakan sebagai host system (sistem utama) untuk menjalankan seluruh pengembangan.

Fungsi utama:

Menjalankan Arduino IDE

Menjalankan Docker Desktop

Menjalankan WSL 2 sebagai jembatan ke Linux

Pastikan:

Windows 11 sudah ter-update

Virtualization aktif di BIOS

RAM minimal 8 GB (disarankan)

2️⃣Mengaktifkan WSL 2 (Windows Subsystem for Linux)

WSL 2 memungkinkan kita menjalankan Linux langsung di dalam Windows tanpa dual boot.

Langkah-langkah:

Buka PowerShell (Run as Administrator)

Jalankan perintah:

wsl --install

Restart komputer


Pastikan WSL versi 2:


wsl --set-default-version 2
3️⃣ Instalasi Ubuntu 24.04 LTS (OS Guest)

Ubuntu digunakan sebagai lingkungan Linux utama untuk ROS 2.


Langkah-langkah:


1. Buka Microsoft Store


2. Cari Ubuntu 24.04 LTS


3. Install dan jalankan


4. Buat username & password Linux

4️⃣ Update Sistem Ubuntu

Sebelum instalasi ROS 2, sistem harus diperbarui.


Kode:


sudo apt update

sudo apt upgrade -y
5️⃣ Instalasi ROS 2 Jazzy Jalisco

ROS 2 berfungsi sebagai middleware komunikasi.


Langkah-langkah umum:

  
1. Tambahkan locale:

  
sudo apt install locales
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
  
2. Tambahkan repository ROS 2:

  
sudo apt install software-properties-common
sudo add-apt-repository universe
  
3. Install ROS 2 Jazzy:

  
sudo apt update
sudo apt install ros-jazzy-desktop -y
  
4. Setup environment:

  
source /opt/ros/jazzy/setup.bash
6️⃣ Instalasi Docker Desktop

Docker digunakan untuk menjalankan micro-ros-agent secara terisolasi.


Langkah-langkah:

    
Download Docker Desktop for Windows

    
Install dan aktifkan:

    
WSL 2 backend
    
Restart jika diminta

7️⃣ Menjalankan micro-ros-agent (Docker)

Micro-ROS Agent berfungsi sebagai jembatan antara mikrokontroler dan ROS 2.


Jalankan container:

    
docker run -it --rm \
  --net=host \
  microros/micro-ros-agent:jazzy \
  serial --dev /dev/ttyUSB0 -b 115200
8️⃣ Instalasi Arduino IDE

Arduino IDE digunakan untuk pemrograman mikrokontroler.


Langkah-langkah:

      
1. Download Arduino IDE

      
2. Install di Windows

      
3. Tambahkan board: Arduino Uno

      
4. Install library: DHT Sensor Library by Adafruit

9️⃣ Instalasi Python (untuk ROS 2 Node)

Python digunakan untuk membuat logic node, monitoring, dan otomasi.


Pastikan Python Terinstall


python3 --version

Jika belun:


sudo apt install python3 python3-pip -y

#include 
#define DHTPIN 2
#define DHTTYPE DHT11
#define BUZZER 8
DHT dht(DHTPIN, DHTTYPE);
void setup(){Serial.begin(9600);dht.begin();pinMode(BUZZER,OUTPUT);} 
void loop(){float h=dht.readHumidity();float t=dht.readTemperature();
if(t<32||t>36||h<50||h>70) digitalWrite(BUZZER,HIGH); else digitalWrite(BUZZER,LOW); delay(2000);}

Troubleshooting

  • Arduino gagal dikompilasi → Periksa library & board
  • Node ROS 2 tidak terhubung → Cek port & micro-ROS agent
  • Perintah ROS 2 tidak dikenali → Source environment ROS 2
